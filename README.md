# Autograd Engine – Credit Risk Studio

A full‑stack demo of a custom autograd engine powering an MLP for credit risk scoring, with a React + Vite visualizer and a Flask API backend. Includes async training with progress updates, inference, explainability, and model artifacts export.

## Features
- Custom autograd engine + MLP training loop
- Async training with job queue + progress bar
- Model inference API
- Lightweight feature contribution explainability
- Model metrics, audit report, and confusion matrix endpoint
- React dashboard for training, inference, and visualization
